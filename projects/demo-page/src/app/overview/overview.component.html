The pr-grid package provides a grid component to implement a table with a robust set of features. and according to 08:15's styleguide

Installation

To install package by npm:
npm install @parlament/grid

To make it work in your project:
1) Import component: 




  2) Use pr-grid component in your html




  3) Give pr-grid a PrGridobject in the [grid] input




PrGrid Object




  The PrGrid object allows you to provide the data and metadata of the table you want to create.





  The grid metadata is separate from the actual data so the grid can be saved in UserPreferences or Presets




  columnGroups - array defining all the currently visible columns.
  

  rows - array of all the rows in the grid.
  

  columnToCellMapper - map of all the possible column names in the grid to a function that returns a PrCell




  the grid runs each row through all visible columns. and gets the cell to show in each column.




  Don't forget to set  --pr-grid-max-width  on the  pr-grid . this should be the size of your container.
